Django Portfolio

A blog web application

  • This is a CRUD portfolio webapp I made to replace this website produced with pelican static site generator as the functions are limited for me.

  • Integrated a Postgres database for production level.

  • Amazon S3 bucket to serve static files.

  • Login as admin to change or add content of webapp. (eg. change introduction, add projects, add blog articles, etc..)

  • Once I have made this project to my liking, this static site will be shut down and replaced the project completly.

  • Planning to integrate React for frontend.

  • Deployed live on Heroku.
