New Taipei City House Prediction

A web app on estimating house price

  • For this project, I used a dataset from 2020 about New Taipei City's real estate dealings from offical Taiwan's government data source (https://data.gov.tw/en).

  • Did feature extraction, feature engineering, data cleaning on the messy dataset with pandas, numpy packages.

  • Data visualization with matplotlib and seaborn packages.

  • Build a linear regression model using scikit-learn with 82% accuracy predicting the price of house based on several features such as location, area, building materials and number of rooms.

  • Deployed on AWS EC2, the web app using html/css/js with Flask as a backend framework and nginx as web server on (ec2-13-250-12-205.ap-southeast-1.compute.amazonaws.com)

  • Planning to keep on working with this project, trying out more machine learning models and showing house location on map in the web app.

  • Sometimes, the web app doesn't work because the aws instance restarts occasionally and I need to go start the flask app manually again. I should probably automate this with gunicorn.
