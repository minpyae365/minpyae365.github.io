Battle of Neighborhoods

A visualization and analysis project on choosing a neighborhood based on someone's interests and concerns.

  • This is a self project about a student trying to choose the best neighborhood for him based on his interests and concerns.
  • Worked with multiple datasets such as New York crime data, foursquare API and New York geojson, to get insights from these data sources.
  • Dealed with json data.
  • Drew data from Foursquare API on data about locations, ratings and venues.
  • Used geocoder and Folium python package to visualize location data on map.
  • Showed clustering and pinpointing visualization on folium map.
